プライバシーポリシー

本アプリ（衛生管理者 爆速合格）は、ユーザーの皆様のプライバシーを尊重し、個人情報の保護に努めています。

1. 収集する情報

本アプリでは、以下の情報を収集する場合があります：

  • アプリ内広告（Google AdMob）: 広告配信の最適化および統計データ収集のため、デバイス識別子などの非個人情報を利用します。
  • 学習進捗: ユーザーの学習進捗データは、デバイス内のローカルストレージ（SharedPreferences）に保存されます。

2. 情報の利用目的

収集した情報は、以下の目的でのみ利用します：

  • 広告の配信およびパーソナライズ
  • アプリの品質向上とバグ修正のための分析

3. 第三者への提供

法令に基づく場合を除き、個人を特定できる情報を第三者に提供することはありません。

4. プライバシーポリシーの変更

本ポリシーの内容は、必要に応じて変更される場合があります。変更後のポリシーは本画面または公式サイトにて公開します。

5. お問い合わせ

プライバシーに関するお問い合わせは、アプリ内のフィードバック機能またはストアのサポートURLよりご連絡ください。
